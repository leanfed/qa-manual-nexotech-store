# Requerimientos

## Requerimientos funcionales

| ID | Requerimiento | Descripción | Prioridad |
|---|---|---|---|
| RF-001 | Visualizar productos | El usuario debe poder ver productos tecnológicos en la página principal. | Alta |
| RF-002 | Buscar productos | El usuario debe poder buscar productos por nombre o descripción. | Alta |
| RF-003 | Filtrar por categoría | El usuario debe poder filtrar productos por categorías. | Alta |
| RF-004 | Ver detalle | El usuario debe poder acceder al detalle de un producto. | Alta |
| RF-005 | Agregar al carrito | El usuario debe poder agregar productos al carrito desde home o detalle. | Alta |
| RF-006 | Modificar cantidad | El usuario debe poder aumentar o disminuir cantidades en el carrito. | Alta |
| RF-007 | Eliminar producto | El usuario debe poder eliminar productos del carrito. | Alta |
| RF-008 | Vaciar carrito | El usuario debe poder vaciar completamente el carrito. | Media |
| RF-009 | Persistencia local | El carrito debe conservar información al recargar la página mediante LocalStorage. | Alta |
| RF-010 | Acceder al panel admin | El administrador debe poder ingresar al panel con credenciales definidas. | Alta |
| RF-011 | Crear producto | El administrador debe poder cargar un nuevo producto. | Alta |
| RF-012 | Editar producto | El administrador debe poder modificar los datos de un producto existente. | Alta |
| RF-013 | Eliminar producto | El administrador debe poder eliminar productos del catálogo. | Alta |
| RF-014 | Restaurar catálogo | El administrador debe poder restaurar los productos iniciales. | Media |

## Requerimientos no funcionales

| ID | Requerimiento | Descripción | Prioridad |
|---|---|---|---|
| RNF-001 | Diseño responsive | La aplicación debe adaptarse correctamente a pantallas móviles. | Alta |
| RNF-002 | Claridad visual | Las tarjetas, botones y textos deben ser legibles y consistentes. | Alta |
| RNF-003 | Rendimiento básico | Las imágenes y recursos deben cargar de forma adecuada en GitHub Pages. | Media |
| RNF-004 | Persistencia local | Los datos almacenados localmente deben recuperarse correctamente. | Media |
