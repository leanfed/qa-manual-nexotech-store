# Publicacion del informe visual

Este repositorio esta preparado para publicar la documentacion visual desde GitHub Pages usando la carpeta `/docs`.

Configuracion recomendada:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`

El archivo principal es `docs/index.html`. La pagina muestra el contenido QA en secciones visuales, por lo que no es necesario abrir los archivos `.md` desde el navegador.
